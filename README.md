# The Infinity Archive

A Grimdark-styled personal reading tracker for the Black Library Grand
Reading Project — a curated journey through modern Warhammer 40,000
fiction (post-Horus Heresy), spanning ~108 novels across 8 thematic
phases.

Single-user app. Built for one reader, tracked for one bookshelf.

## Stack

- **Frontend:** React 19, Tailwind CSS, shadcn/ui, Framer Motion
- **Backend:** Supabase (planned — Sprint B)
- **Hosting:** Vercel (planned — Sprint B)
- **Current state:** Frontend-only with localStorage persistence
